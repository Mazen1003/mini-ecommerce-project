# Mini E-Commerce Prototype
## Problem Statement
Små lokala butiker har ofta svårt att skapa en enkel digital närvaro där kunder kan se produkter online.

Existerande e-handelssystem är ofta för komplexa och dyra för små verksamheter.

Detta projekt syftar till att modellera och prototypa en enkel e-handelslösning med produkter, kategorier och en varukorg.

## Stakeholders

**Customer**
- Bläddrar produkter
- Ser produktdetaljer
- Lägger produkter i varukorg

**Admin**
- Skapar nya produkter
- Organiserar produkter i kategorier

**Guest**
- Kan se produkter men inte genomföra köp
